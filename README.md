# superdefi
présentation du projet :
Ce projet consiste à développer un site web collaboratif en équipe.
Il servira de support pour pratiquer GitHub, le travail en équipe, 
une architecture de projet claire, et la séparation entre développement créatif et développement fonctionnel 

Vous êtes obligés de suivre l’architecture, les noms de fichiers et les classes CSS définies ici afin d’éviter les erreurs,
les doublons, les conflits Git et les pertes de temps inutiles.

architecture pour le site :
/projet-site/
/assets/
     /css/
       style.css
     /js/

/includes/
    header.php
      footer.php

  Classes CSS à utiliser :
.section	Bloc principal d'une section
.section-title	Titre de section
.section-text	Paragraphe de sectionfunctions.php
.navbar	Barre de navigation
.nav-link	Liens du menu
.logo	Logo ou nom du site


CheckLIST :

- Accueil (accueil.php)
- À propos (about.php)
- Contact (contact.php)

- Header avec navigation (menu + logo)
- Footer avec liens utiles
- Fonctions PHP partagées (functions.php)
- Fichiers CSS + assets
