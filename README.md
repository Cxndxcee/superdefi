# superdefi
Ce projet consiste à développer un site web collaboratif en équipe.
Il servira de support pour pratiquer GitHub, le travail en équipe, 
une architecture de projet claire, et la séparation entre développement créatif et développement fonctionnel 

Vous êtes obligés de suivre l’architecture, les noms de fichiers et les classes CSS définies ici afin d’éviter les erreurs,
les doublons, les conflits Git et les pertes de temps inutiles.
